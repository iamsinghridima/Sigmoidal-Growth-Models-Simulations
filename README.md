# Sigmoidal Growth Models Simulations

A comprehensive model implementation of five classic sigmoidal growth models used in microbiology, ecology, and population dynamics.

## Models Implemented

1. **Logistic Model** - Classic S-curve model describing population growth with carrying capacity
   - $N(t) = \frac{K}{1 + e^{-r(t-t_0)}}$

2. **Gompertz Model** - Asymmetric growth model useful for aging and tumor growth
   - $N(t) = K e^{-e^{-r(t-t_0)}}$

3. **Baranyi-Roberts Model** - Mechanistic model with lag phase, particularly for microbial growth
   - Includes lag time and acceleration phase parameters

4. **Richard's Model** - Generalized logistic model with flexibility parameter
   - Reduces to logistic model when θ=1

5. **Von Bertalanffy Model** - Growth model based on anabolic and catabolic processes
   - $N(t) = N_\infty (1 - e^{-K(t-t_0)})^{1/3}$

These scripts calculates growth values, prints simulation tables, and generates a combined plot comparing population size and growth rate over time.

## Features

- Continuous curve generation for smooth visualization
- Discrete point calculations for data fitting
- Population growth rate computations (dN/dt)
- Console output table with population dynamics
- Multi-panel visualization comparing all models
- Twin-axis plots showing both population and growth rates

## Files

- `sigmoidal_growth_models.py` - Python script for simulating and plotting the models.
- `sigmoidal_growth_models_plot.png` - Output plot generated by the python script.
- `sigmoidal_growth_models.m` - MATLAB script for simulating and plotting the models.
- `sigmoidal_growth_models.xlsx` - Excel Spreadsheet for simulating and plotting the models.

The scripts will:
1. Generate a console table showing time, population values, and growth rates (t = 0 to 30)
2. Produce a comprehensive figure with:
   - Top panel: overlay of all 5 models
   - Individual panels: each model with twin-axis growth rate visualization
   - Comparison panel: growth rate dynamics across models

## Parameters

Default parameters are configured in the script and can be easily modified:

- **Carrying capacity (K)**: 1000 cells/organisms
- **Growth rate (r/μ)**: 1.0
- **Initial population (N₀)**: 5 cells/organisms
- **Time range**: 0 to 30 time units
- **Resolution**: 2000 points for continuous curves

## Mathematical Background

These models describe population dynamics where growth follows an S-shaped curve, characterized by:
- **Lag phase**: Initial slow growth (variable across models)
- **Log phase**: Exponential growth period
- **Stationary phase**: Growth plateau approaching carrying capacity
