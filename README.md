# Sigmoidal Growth Models Simulations

This repository contains Python simulations and visual comparisons of five sigmoidal growth models:

- Logistic model
- Gompertz model
- Baranyi-Roberts model
- Richard's model
- Von Bertalanffy model

The script calculates growth values, prints simulation tables, and generates a combined plot comparing population size and growth rate over time.

## Files

- `sigmoidal_growth_models.py` - Python script for simulating and plotting the models.
- `sigmoidal_growth_models_plot.png` - Output plot generated by the script.
